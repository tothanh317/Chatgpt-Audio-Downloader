# Chatgpt-Audio-Downloader Extention
Download audio from ChatGPT with flexible voice switching and bulk download support. 

Chromewebstore: https://chromewebstore.google.com/detail/akobaofghpokhcnglkojhaednbcpjohd?utm_source=item-share-cb
ChatGPT Audio Downloader is a powerful browser extension that allows you to download Text-to-Speech (TTS) audio files from ChatGPT responses with a single click. Featuring a modern interface, it is perfect for language learners, audiobook creators, lecture savers, and anyone looking to listen to ChatGPT conversations offline.

Key Features:
- Download high-quality audio files (MP3 or AAC formats).
- Bulk Download Support: Select multiple messages at once and download them bundled in a ZIP archive.
- Personalized Voices: Customize the speech voice for each individual message (Cove, Ember, Arbor, Breeze, Vale, etc.).
- Random Silence Insertions: Insert smart pause delays between merged audio files.
- Seamless Sidebar Interface integrated directly into the ChatGPT website.

How to Use:
1. Open ChatGPT (https://chatgpt.com).
2. Click the extension icon or trigger button to open the sidebar.
3. Choose your preferred audio format and speech voice.
4. Click the "Download" button under any Assistant response, or select multiple messages to download in bulk via the sidebar.
